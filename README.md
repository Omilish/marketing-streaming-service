## Онлайн-кинотеатр, предоставляющий к просмотру как фильмы и сериалы, так и мультфильмы, провёл серию рекламных кампаний и теперь хочет оценить, какой вклад в итоговый трафик они внесли.


### Стэк: Pyhton (pandas, seaborn, scipy, statsmodels, sklearn, factor_analyzer)

### Выявить:

- Количество уникальных пользователей, пришедших по рекламным каналам в разрезе этапов воронки продаж

- Фродовые данные в каналах.

- Расчет показателей рентабельности каналов привлечения

### Методы: визуализация, расчет статистик и метрик, факторный анализ

### Вывод:

- Оценка конверсии для каждого канала с точки зрения привлечения потенциальных клиентов

- Выявлены этапы с наибольшим числом фродовых кликов

- Рассчитаны метрики окупаемости каналов
